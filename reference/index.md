# Package index

## Core download function

Main workhorse — downloads, processes and saves datasets from Kenneth
French’s data library.

- [`FFdownload()`](https://www.sebastianstoeckl.com/ffdownload/reference/FFdownload.md)
  : Downloads Datasets from Kenneth French's Website

## Convenience functions

New in v1.2.0 — simpler access for interactive and exploratory use.

- [`FFget()`](https://www.sebastianstoeckl.com/ffdownload/reference/FFget.md)
  : Download and return a single French dataset directly
- [`FFlist()`](https://www.sebastianstoeckl.com/ffdownload/reference/FFlist.md)
  : List available datasets on Kenneth French's website
- [`FFmatch()`](https://www.sebastianstoeckl.com/ffdownload/reference/FFmatch.md)
  : Preview fuzzy-matching results before downloading
